<ordered list>
  1.객체 : 객체 지향 프로그램의 대상
  2.클래스 : 객체를 프로그래밍 하기 위해 코드로 정의해놓은 상태
  3.인스턴스 : new 키워드를 사용하여 클래스를 메모리에 생성한 상태
  4.멤버변수 : 클래스의 속성, 특성
  5.메서드 : 멤버변수를 이용하여 클래스의 기능을 구현한 함수
  6.참조변수 : 메모리에 생성된 인스턴스를 가리키는 변수
  7.참조값 : 생성된 인스턴스의 메모리 주소 값
  8.매개변수(파라미터) : 메서드 수행에 필요한 입력값을 저장하는 변수
